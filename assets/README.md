# Assets Directory

This directory is intended for storing images and icons for the e-commerce site.

## Usage

Place your product images, logo, and other static assets here.

### Image Recommendations

- **Product Images**: 400x400px or larger, square format
- **Logo**: SVG format preferred, or PNG with transparency
- **Icons**: SVG format for best quality at any size

### Supported Formats

- JPG/JPEG - for photos
- PNG - for images with transparency
- SVG - for logos and icons
- WebP - for optimized web images

## Note

The current products.json uses Unsplash images as placeholders. You can replace them with your own images stored in this directory.

Example usage in products.json:
```json
{
    "image": "assets/product-1.jpg"
}
```
